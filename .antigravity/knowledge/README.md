# Mad Lab Aurora Knowledge Base

Este diretório serve como o repositório central de conhecimento para os Agentes de IA dentro do projeto Certum.

## Bibliotecas Documentadas

Atualmente, buscamos integração com as seguintes bibliotecas fundamentais:

- **GSAP:** Animações e Motion Design.
- **Tailwind CSS v4:** Estilização e Design System.
- **Shadcn/UI:** Componentes de interface.
- **Framer Motion:** Micro-interações.
- **Vercel AI SDK:** Integração de IA e Generative UI.
- **The Book of Shaders:** Conhecimento avançado de GLSL/GPU.

## Estrutura

O conhecimento é trazido da pasta local de documentações via Junctions (links simbólicos de diretório) ou cópia direta no Windows para garantir que a IDE possa indexar e consultar esses arquivos em tempo real durante o desenvolvimento.
