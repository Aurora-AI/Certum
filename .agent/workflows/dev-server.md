# Dev Server

Use este workflow para iniciar o ambiente de desenvolvimento.

// turbo-all

## Steps

1. **Instale** dependências (se necessário):

```bash
npm install
```

2. **Inicie** o servidor de desenvolvimento:

```bash
npm run dev
```

3. **Aguarde** o servidor estar disponível (geralmente porta 5173 para Vite).

4. **Informe** ao usuário que o servidor está rodando e forneça a URL local.
